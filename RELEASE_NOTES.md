# AudioReactiveImagePOC v1.0.0

First shareable proof-of-concept release.

## Included Formats

- VST3: `AudioReactiveImagePOC.vst3`
- AU macOS: `AudioReactiveImagePOC.component`
- Standalone macOS app: `AudioReactiveImagePOC.app`

## What It Does

- Passes stereo audio through without intentional modification.
- Measures global RMS, peak, and a smoothed audio envelope.
- Lets the user upload a PNG/JPG/JPEG image.
- Animates the image scale, rotation, and opacity/brightness from audio energy.
- Saves plugin parameters and the local image path in DAW state.

## Install macOS

Copy the VST3 to:

```txt
~/Library/Audio/Plug-Ins/VST3/
```

Copy the AU to:

```txt
~/Library/Audio/Plug-Ins/Components/
```

Then rescan plugins in your DAW.

## Known Limitations

- V1 saves only the image path, not embedded image bytes.
- If the image is moved or deleted, the plugin shows a missing-file message.
- AU host discovery may require restarting the host or AudioComponent registrar.
- No FFT, beat detection, or frequency-band animation in V1.
- No notarized macOS distribution package yet.

## Build Provenance

Built from JUCE/CMake sources in this repository. Release ZIPs generated by `scripts/package-macos-release.sh` are not committed to Git.
